How to generate a Gitleaks JSON report

Run gitleaks against your repo and export the report in JSON format. Use detect for a clean tree scan, or scan the full git history (recommended for finding committed secrets):

# scan working directory
gitleaks detect --source . --report-format json --report-path leaks.json --no-banner

# scan full git history (recommended)
gitleaks detect --source . --log-opts="--all" --report-format json --report-path leaks.json --no-banner

# scan a specific branch / from a commit
gitleaks detect --source . --log-opts="main..HEAD" --report-format json --report-path leaks.json

Then switch to the Import tab and drop leaks.json or paste its contents.
